# Student Coffee Report

Этот проект собирает данные о подготовке студентов к экзаменам и формирует отчёты по расходам на кофе.  
Скрипт читает CSV-файлы с данными студентов и выводит медианную сумму трат на кофе для каждого студента.  

## Примеры запуска

Все примеры ссылаются на тестовые CSV-файлы, указанные в ТЗ.  
Результаты работы скрипта представлены в виде таблиц.

Скриншоты работы находятся в папке `screenshots/`:

- `report_math.png` — отчёт по файлу `math.csv`  
- `report_physics.png` — отчёт по файлу `physics.csv`
- `report_programming.png` — отчёт по файлу `physics.csv`    
- `report_all.png` — агрегированный отчёт по всем файлам  

Скрипт поддерживает несколько файлов одновременно и формирует отчёт по всем переданным данным.
